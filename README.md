# FPGA Dual-Mode Gaming System

A custom, hardware-accelerated Dual-Mode Reaction Timer and Memory Game implemented in Verilog for the Basys 3 Artix-7 FPGA.

## 🚀 Features
* **Mode 1:** Sequential Memory Game with pseudo-random hardware sequences.
* **Mode 2:** Precision Reaction Timer measuring reflex limits down to 1ms.
* **Hardware:** Driven by custom clock dividers, an 8-bit LFSR, and debounced physical inputs.
